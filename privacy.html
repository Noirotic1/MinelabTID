Privacy Policy

Target ID · Effective June 9, 2026

Target ID is a free reference tool for metal detectorists, developed by Mike Noirot. This policy describes what data the app handles — which is, in short, none of yours.

Data We Collect

Target ID does not collect, transmit, store, sell, or share any personal data. There are no accounts, no sign-ins, no analytics, no advertising, no tracking, and no third-party SDKs.

Data Stored on Your Device

Your preferences stay on your device and never leave it:

  • Display settings (theme, font size, interface mode)
  • Your configured detector machine slots and any custom target cards
  • App files cached for offline use

This data is stored in your browser's local storage on your device only. Uninstalling the app or clearing site data removes it completely. The optional Backup feature saves a settings file to a location you choose; that file is created locally and is never sent anywhere.

Network Use

The app connects to the internet only to load and update itself from this website. No usage information is sent with those requests beyond what any web page load involves.

External Links

The app contains links to external sites, such as the developer's Ko-fi page and the data sources credited in the app. Those sites have their own privacy policies, and this policy does not cover them.

Children

Target ID does not collect data from anyone, including children.

Changes to This Policy

If this policy ever changes, the updated version will be posted at this address with a new effective date.

Contact

Questions about this policy or the app: noirot@gmail.com
